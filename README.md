# Calculadora de Comissão do Vendedor – Python + Shell Script

## Descrição
Projeto desenvolvido para calcular a comissão de vendas usando **Python**, com automação em **Shell Script (Linux)** para facilitar a execução.

---

## Arquivos
- `calculadora_comissao_vendedor.py` → código principal em Python  
- `calculadora_comissao_vendedor.sh` → script que executa o Python automaticamente  
- `comandos.txt` → lista de comandos usados no terminal Linux  
- `README.md` → documentação do projeto
